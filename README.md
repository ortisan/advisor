# Advisor

Project to get advices from Assets, using microservices.


### Services and port numbers

| Service          | Port Number     |
|----------------  |---------------- |
| Config Server    |            8888 |
| Discovery Server |            9777 |
| Advisor Service  |            9888 |


### Start 

```sh
docker-compose up --build
```
